<h>PaperFeed</h>
AI slopped infinite scrolling feed  website to read preprint papers during a break. Heavily inspired by https://github.com/Miguel07Alm/arxivtok .

Use vite for frontend and minishlab (https://github.com/MinishLab/model2vec) small models for sorting and filtering papers. 
At the moments code supports arxiv API, and chemrxiv and biorxiv through crossref API.
Additional repositories are easy to add through crossref prefixes.
